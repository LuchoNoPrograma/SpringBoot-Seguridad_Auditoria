SUBJEFE DE PROGRAMA

Solo pueden ingresar aquellos usuarios que tienen el rol asociado de 'SUBJEFE_PROGRAMA'
El usuario 1(jefesistemas): tiene todos los roles/autoridades y accede a esta pagina por la anotacion @PreAuthorize de su controlador
El usuario 2(subjefe): tiene el rol para acceder a esta pagina por la anotacion @PreAuthorize de su controlador

Nota.- No es necesario poner ROLE_SUBJEFE_PROGRAMA
con tal de que hayas establecido ROLE_SUBJEFE_PROGRAMA en la DB
Por lo tanto cuando se use hasRole() o hasAnyRole(), escribir en parametro el nombre del rol sin prefijo

@PreAuthorize("hasAnyRole('SUBJEFE_PROGRAMA')")
public String cualquieraQueTengaRolCoordinador(){
return "esta-pagina";
}
hasAnyRole hace lo siguiente:
Si el usuario autenticado tiene ALGUN rol("SUBJEFE_PROGRAMA"){
Acceso al metodo
}
