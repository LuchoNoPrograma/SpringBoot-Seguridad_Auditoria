COORDINADOR_PROGRAMA

Solo pueden ingresar aquellos usuarios que tienen el rol asociado de 'COORDINADOR_PROGRAMA'
El usuario 1(jefesistemas): tiene todos los roles/autoridades pero accede a esta pagina por la anotacion @PreAuthorize de su controlador
El usuario 2(coordinador): tiene el rol para acceder a esta pagina por la anotacion @PreAuthorize de su controlador

Nota.- No es necesario poner ROLE_COORDINADOR_PROGRAMA
con tal de que hayas establecido ROLE_COORDINADOR_PROGRAMA en la DB
Por lo tanto cuando se use hasRole() o hasAnyRole(), escribir en parametro el nombre del rol sin prefijo

@PreAuthorize("hasAnyRole('COORDINADOR_PROGRAMA')")
public String cualquieraQueTengaRolCoordinador(){
return "esta-pagina";
}
hasAnyRole hace lo siguiente:
Si el usuario autenticado tiene ALGUN rol("COORDINADOR_PROGRAMA"){
Acceso al metodo
}
